Tentang Template Website Catalog-Z

Anda dapat mendukung website TemplateMo dengan memberikan donasi kecil melalui PayPal. Ini akan sangat membantu kami. Kami harap Anda menyukai template foto/video Catalog-Z ini. Kami rutin membuat template baru untuk Anda. Silakan kunjungi kembali website TemplateMo.

Kredit diberikan kepada Pexels dan Unsplash untuk foto dan video yang digunakan. Catalog-Z adalah template HTML gratis berbasis Bootstrap 5 untuk website foto dan video.

Anda diperbolehkan menggunakan template ini untuk website komersial maupun non-komersial. Anda dapat mengintegrasikannya dengan CMS apa pun. Namun, Anda TIDAK diperbolehkan mendistribusikan ulang file ZIP template ini di situs koleksi template. Silakan hubungi kami untuk informasi lebih lanjut. Terima kasih.
